
# jirer

## Quickstart

Install using pip:

```
pip install git+https://github.com/lucasea777/jirer.git --user
```

Ensure that `~/.local/bin` is in PATH, otherwise add something like:

```
PATH=PATH:/home/$(whoami)/.local/bin
```

to `~/.zshrc` or `~/.bashrc`
